# v2.ext.ggt1024.com

本项目为伽马影迷插件文档源码，用 `VuePress` 生成静态网站内容。

## 使用

推荐使用包管理器器`yarn`，推荐 node.js 版本`v14.0.0`以上

安装相关依赖

```bash
yarn install
```

本地开发

```bash
yarn docs:dev
```

打包

```bash
yarn docs:build
```

For local development, [yarn](https://yarnpkg.com/) is preferred as package manager:

```bash
yarn docs:dev
```
